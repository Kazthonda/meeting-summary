【コード例】

GPU 情報の取得:

  const GPUDetector = require('./lib/gpu-detector');
  const detector = new GPUDetector();
  const gpuInfo = detector.detect();
  
  console.log(gpuInfo.gpuType);      // 'nvidia', 'metal', etc
  console.log(gpuInfo.isAvailable);  // true/false
  console.log(gpuInfo.device);       // 'cuda', 'mps', etc

キャッシュの管理:

  const CacheManager = require('./lib/cache-manager');
  const cache = new CacheManager();
  
  // キャッシュに保存
  cache.set(filePath, { text: '...' });
  
  // キャッシュから取得
  const cached = cache.get(filePath);
  
  // 統計情報
  const stats = cache.getStats();

【技術仕様】

GPU 検出:
  - NVIDIA: nvidia-smi コマンドで検出
  - Apple: sysctl で CPU ブランド確認
  - AMD: rocm-smi コマンドで検出

キャッシング:
  - ハッシュアルゴリズム: SHA-256
  - キャッシュ保存先: .cache/ ディレクトリ
  - TTL: 24時間（自動削除）

faster-whisper:
  - Python ブリッジで実行
  - 自動デバイス選択
  - 量子化対応（int8, float16, float32）

【セキュリティ考慮事項】

✓ キャッシュディレクトリは自動作成
✓ ハッシュ値はファイルのみから計算
✓ キャッシュファイルは JSON 形式
✓ キャッシュは 24 時間で自動削除
