【処理フロー】
  iPhone音声(mp3/m4a) 
    ↓
  Web UI (HTML/CSS/JS - ドラッグ&ドロップ)
    ↓
  Express API (Node.js)
    ├→ OpenAI Whisper API (音声認識)
    └→ Anthropic Claude API (議事録生成)
    ↓
  ファイル変換 (Markdown → Word/PDF)
    ↓
  ユーザーダウンロード

【主要機能】
  🎯 GCSM IAMaaS ガイドライン 100% 対応
  📱 iPhone対応のレスポンシブUI
  🔄 3段階の出力形式 (Markdown / Word / PDF)
  ⚡ 自動クリーンアップとセキュアファイル処理
  📊 詳細なプログレス表示とエラーハンドリング

【セットアップ】
  1. cp .env.example .env
  2. APIキー設定 (OpenAI + Anthropic)
  3. npm install
  4. npm start
  5. http://localhost:3000

【デプロイオプション】
  ✓ ローカル実行 (npm start)
